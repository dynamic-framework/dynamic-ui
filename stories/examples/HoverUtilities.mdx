# Hover-prefixed utilities

This project extends Bootstrap utilities to support Tailwind-like `hover:` variants (e.g., `hover:shadow-sm`, `hover:bg-primary-100`, `hover:text-danger`).

## Shadows

<Unstyled>
  <ExampleHoverShadowBoxes />
</Unstyled>

<Source
  code={`
<div class="p-4 rounded border shadow-none hover:shadow-sm">...</div>
<div class="p-4 rounded border shadow-none hover:shadow">...</div>
<div class="p-4 rounded border shadow-none hover:shadow-lg">...</div>
<div class="p-4 rounded border shadow-sm hover:shadow-none">...</div>
  `}
  language="html"
/>

## Text color variants

Palette-level variants (e.g., `hover:text-primary-100`, `hover:text-danger-200`) are supported on top of the base `hover:text-*` classes.

<Unstyled>
  <ExampleHoverTextColorVariants />
</Unstyled>

<Source
  code={`
<span class="hover:text-primary-100">hover:text-primary-100</span>
<span class="hover:text-primary-300">hover:text-primary-300</span>
<span class="hover:text-primary-700">hover:text-primary-700</span>
<span class="hover:text-danger-200">hover:text-danger-200</span>
<span class="hover:text-success-400">hover:text-success-400</span>
<span class="hover:text-warning-500">hover:text-warning-500</span>
  `}
  language="html"
/>

## Border color variants

Palette-level variants (e.g., `hover:border-primary-100`, `hover:border-danger-200`) are supported by combining the `hover:` prefix with any palette shade available in the design system.

<Unstyled>
  <ExampleHoverBorderColorVariants />
</Unstyled>

<Source
  code={`
<div class="border border-2 hover:border-primary-100">...</div>
<div class="border border-2 hover:border-primary-300">...</div>
<div class="border border-2 hover:border-danger-200">...</div>
<div class="border border-2 hover:border-success-400">...</div>
  `}
  language="html"
/>
