  ### Foundations

  Dynamic Framework is based on Bootstrap 5, with a standardization layer that unifies color tokens, typography, spacing, and utility variants (hover:, dark:) to deliver consistent, accessible interfaces. We extend Bootstrap with additional utilities and responsive variants to accelerate UI building. Examples include expanded spacing (0–30), extended grids (g-col-13…24), typography utilities (fs-body, display-*), overlay opacity, text truncation, text-wrap, vertical align, responsive width/height, and more.

  <br />
  #### Typography
  - Default font family: Jost, optimized for UI legibility
  - Hierarchy: Headings (h1–h6), Display (`.display-1` … `.display-6`), Font-size utilities (`.fs-1` … `.fs-6`, `.fs-body`)

  See: <a href="/?path=/story/design-system-foundations-typography">Design System/Foundations/Typography</a>.

  <br />
  #### Colors (Semantic states)
  Use semantic color tokens for clarity and accessibility:
  - Primary, Success, Warning, Danger, Info, Secondary, Light, Dark
  - Text: `.text-{token}`, Background tints: `.bg-{token}-{25|50|100|200|300|400|500|600|700|800|900}`, Border: `.border-{token}`
  - Light variants are intended for use on dark backgrounds to ensure contrast.

  Leverage Bootstrap grid: Container (.container), Rows (.row), Columns (.col, .col-6, .col-md-4 …). You can use the classic <code>&lt;div class="row"&gt;</code> approach, or our CSS Grid-based column system via <code>&lt;DLayout /&gt;</code>.

  ### Accessibility guidance
  - Use semantic HTML (headings, lists, buttons, links) for structure
  - Ensure WCAG AA contrast for text/background (e.g., danger on light bg)
  - Preserve focus states and keyboard navigation (`:focus-visible`)
  - Provide clear feedback: success, warning, danger, info colors
  - Prefer motion small/medium durations (150–250ms) for UI transitions
