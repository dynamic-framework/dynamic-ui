  # Dynamic Framework [![npm](https://img.shields.io/npm/v/@dynamic-framework/ui-react?style=flat-square)](https://www.npmjs.com/package/@dynamic-framework/ui-react)

  Dynamic Framework components come with production-ready code, design guides, and an active community of financial experience experts.

  Accessing the [main site](https://dynamicframework.dev/) provides additional information and resources.

  The [Github Repository](https://github.com/dynamic-framework/dynamic-ui) is available for review and collaboration.

  ### Getting started with templates

  Modyo provides a base of templates with the framework configured ready to use. The simplest is ```dynamic-react-base-template```, it is recommended to download this template using the [CLI](https://docs.modyo.com/es/platform/channels/cli.html). For further exploration and contribution, you can visit the [GitHub repository.](https://github.com/dynamic-framework/dynamic-react-base-template)

  ```shell
  npx @modyo/cli@latest get dynamic-react-base-template
  ```
