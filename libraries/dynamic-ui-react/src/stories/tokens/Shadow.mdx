# Shadow

Shadows hold important visual clues about objectâ€™s depth and directional movement. The elevation and importance of an object determines the appearance of its shadow.

In the physical world, objects can stack or stick to each other, but they cannot pass through each other. Objects also cast shadows and reflect light.

To understand in more detail review the following documentation : [Bootstrap Shadow](https://getbootstrap.com/docs/5.3/utilities/shadows/)
