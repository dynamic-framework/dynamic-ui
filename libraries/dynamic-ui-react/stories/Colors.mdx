# Colors

To understand in more detail the classNames for colors review the following documentation:

+ [Bootstrap Customize Colors](https://getbootstrap.com/docs/5.2/customize/color/)
+ [Bootstrap Color & Background](https://getbootstrap.com/docs/5.2/helpers/color-background/)
+ [Bootstrap Utility Colors](https://getbootstrap.com/docs/5.2/utilities/colors/)

## Base

Colors can be used with bootstrap [utility classes](https://getbootstrap.com/docs/5.2/utilities/colors/), like:
<span className="bg-black text-white">bg-black text-white</span>

<ColorPalette>
  <ColorItem
    title="--bs-black"
    subtitle="--bs-black-rgb"
    colors={{
      black: 'var(--bs-black)',
    }}
  />
  <ColorItem
    title="--bs-white"
    subtitle="--bs-white-rgb"
    colors={{
      white: 'var(--bs-white)',
    }}
  />
</ColorPalette>
