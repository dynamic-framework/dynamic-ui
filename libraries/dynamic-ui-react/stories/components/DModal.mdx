# Modal

This section provides guidance specifically for using `DModal` in **inline** scenarios.

<DAlert soft type="warning" className='py-1 mb-8'>
  To achieve the behavior of a modal it is necessary to use the **`DContextProvider`** and the **`useDPortalContext`** hook.
  For detailed guidance on the **correct usage** of modals, please refer to the [Modal pattern section](/docs/design-system-patterns-modal--docs) in our documentation.
</DAlert>
