# Offcanvas

This section provides guidance specifically for using `DOffcanvas` in **inline** scenarios.

<DAlert soft type="warning" className='py-1 mb-8'>
  To achieve the behavior of a offcanvas it is necessary to use the **`DContextProvider`** and the **`useDPortalContext`** hook.
  For detailed guidance on the **correct usage** of offcanvas, please refer to the [Offcanvas pattern section](/docs/design-system-patterns-offcanvas--docs) in our documentation.
</DAlert>
