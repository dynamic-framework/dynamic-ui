# Border

Bootstrap includes border utilities to quickly style the **border** and **border-radius** of an element. Great for images, buttons, or any other element.

To understand in more detail review the following documentation : [Bootstrap Border](https://getbootstrap.com/docs/5.3/utilities/borders/#border)
